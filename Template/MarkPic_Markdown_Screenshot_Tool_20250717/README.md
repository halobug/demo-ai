# 🖼️ MarkPic - Markdown 可视化截图工具

**MarkPic** 是一款将 Markdown 内容实时预览为 HTML，并生成高清图片的可视化工具，适用于以下场景：

* 📝 技术文档与学习笔记分享
* 📄 博客文章视觉展示
* 📢 社区海报与宣传内容生成
* 🧾 高质量排版导出图片

## ✨ 功能亮点

* 实时 Markdown 渲染预览
* 支持自定义主题与排版样式
* 一键导出高清图片（适配社交媒体/博客）
* 响应式设计，适配不同设备
* 零依赖、开箱即用的 Web 工具
